# tele-create2 #

### What is this repository for? 
To controll iRobot create2 using peer JS

### How do I get set up? ###

```sh
$ git clone https://github.com/cdra0624/telecreate2.git
$ cd telecreate2
$ npm install
$ npm start
```
browse [localhost:](http://localhost:3333)
